# Datadog Studies

Repositório com projetos práticos de estudo e implementação do **Datadog** — plataforma de observabilidade que unifica monitoramento, métricas, logs e rastreamento de aplicações em um único lugar.

## O que é o Datadog?

Datadog é uma plataforma de **observabilidade em nuvem** que permite:

- **Coletar métricas** de aplicações, infraestrutura e serviços externos
- **Centralizar logs** de todos os seus sistemas em tempo real
- **Rastrear requisições** de ponta a ponta com distributed tracing (APM)
- **Criar dashboards** personalizados para visualizar o estado dos seus sistemas
- **Configurar alertas** para ser notificado quando algo sair do normal

## Por que usar?

Em ambientes modernos com múltiplos serviços, containers e clouds, é difícil saber o que está acontecendo sem uma ferramenta dedicada. O Datadog resolve isso centralizando tudo e correlacionando métricas, logs e traces em uma única plataforma.

## Tecnologias presentes neste repositório

- **DogStatsD** — envio de métricas customizadas via UDP
- **Docker** — todos os projetos rodam em containers
- **Node.js / Python / Oracle** — diferentes linguagens e integrações com o Datadog Agent
